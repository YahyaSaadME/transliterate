Platform and Language Selection

Windows Content

To enable virtual keyboard follow the steps below:

    ➊Go to start

    ➋Then search for settings

    ➌Search for Accessibility

    ➍Head to keyboard

    ➎Toggle on this option:

    Install Hindi and Tamil under language and region tab

Mobile Content

Follow the steps to change the keyboard:

    Head to your Keyboard settings

    Search for languages

    Search and add Hindi
